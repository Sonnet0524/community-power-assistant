> 🐛 项目问题跟踪

## 使用说明

本目录用于存放项目问题和任务：
- Bug 报告
- 功能请求
- 改进建议
- 待办事项

**命名规范**：
- `{类型}-{编号}.md`
- 示例: `bug-001.md`, `feature-002.md`

**类型**：
- `bug` - Bug 报告
- `feature` - 功能请求
- `improvement` - 改进建议
- `task` - 任务
