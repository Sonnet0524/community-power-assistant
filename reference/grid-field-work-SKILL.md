# 国家电网客户经理现场工作指引

## 触发条件

当用户说以下任一触发词时启动本技能：
- "开始"
- "开始现场工作"
- "开始勘察"
- "启动现场工作"

## 工作流程

### 第一阶段：初始化

1. 提示用户输入小区名称，默认查询范围为**四川省成都市**，如需查询其他城市可补充说明城市信息
2. 自动调用 **百度地图 PlaceSearch API** 查询小区地址：

   **API 调用方式**：
   ```bash
   curl "https://api.map.baidu.com/place/v2/search?query=${小区名称}&region=${城市}&city_limit=true&output=json&ak=${BMAP_JSAPI_KEY}"
   ```

   **参数说明**：
   - `query`: 小区名称（用户输入）
   - `region`: 查询城市（默认：成都）
   - `city_limit`: 限制在城市内搜索
   - `ak`: 百度地图密钥（从环境变量 BMAP_JSAPI_KEY 获取）

   **返回结果解析**：
   ```json
   {
     "results": [
       {
         "name": "小区名称",
         "address": "详细地址",
         "province": "四川省",
         "city": "成都市",
         "area": "武侯区",
         "location": {"lng": 104.123, "lat": 30.456},
         "street_id": "街道ID"
       }
     ]
   }
   ```

   **字段提取映射**：
   - 小区名称 → `results[0].name`
   - 小区地址 → `results[0].address`
   - 所属街道 → `results[0].area` + 街道信息
   - 所属社区 → 从 `results[0].address` 解析
   - 经纬度 → `results[0].location.lng`, `results[0].location.lat`
3. 地址确认流程：
   - 调用地点检索接口后最多返回3个匹配度最高的小区地址，按序号列表展示
   - 用户输入对应序号选择地址，直接使用选中的地址
   - 如所有结果都不匹配，用户输入"0"手动输入地址或补充城市信息后重新查询
4. 从选中的POI信息中自动提取并预填小区名称、小区地址、所属街道、所属社区字段
5. 记录当前时间
6. 初始化工作检查清单

#### 5. 用电检查隐患台账（台区巡视）

按以下三个区域逐项检查，拍照记录问题，结果填入巡视记录表：

**A. 低配室**

| 序号 | 检查内容 | 巡视标准 |
|------|----------|---------|
| 1 | 变压器本体 | 核对小区变压器数量、容量、重过载情况；是否有未装表情况 |
| 2 | 低压柜、电容柜 | 标识牌齐全清晰；柜体固定牢固、接地良好；分合闸指示器、电源指示灯、仪表显示正常；开关外观完好、无异味发热；电容器外壳无膨胀变形渗油 |
| 3 | 低配室环境 | 门窗照明完好，无渗漏水，通风良好；消防灭火设备良好数量足够；防小动物措施到位 |

**B. 竖井区域**（酌情抽查，每栋楼需有抽查点位）

| 序号 | 检查内容 | 巡视标准 |
|------|----------|---------|
| 1 | 竖井电缆 | 电缆本体无明显变形，外护套无破损龟裂；密集型母线无弯曲变形浸水，插接处连接可靠 |
| 2 | 表箱 | 无破损集尘；接地可靠无松动；进线漏保开关定值正确（100mA 0.2s 或 300mA 0.2s）；表后漏保已安装且定值正确（30mA 0.1s）；箱内布电线接头连接可靠无明显锈蚀发热 |
| 3 | 竖井环境 | 防火措施完好（防火槽盒/防火毯/防火封堵）；垂直电缆固定良好；无堆积杂物易燃物；强弱电不共井，或共井时强弱电分布两侧且弱电敷设在金属槽盒内、无私拉乱接；竖井内无蓄电池等其他风险源 |

> **注意**：竖井区域不满足要求时，须告知义务督促物业整治，并及时报备（供电所定期向街道、社区报备；每月汇总至中心，由中心统一抄送经信、住建与消防，需附照片）。

**C. 居民充电桩**

| 序号 | 检查内容 | 检查标准 |
|------|----------|---------|
| 1 | 充电桩开关排查（23年安装批次） | 充电桩表箱厂家是否为"山东锐驰电气"；对应开关是否为"JULAN"牌（该批次疑似质量问题，需先摸排情况） |

- **记录内容**：最近一次用电检查时间（年月日）、发现安全隐患处数、已整改处数、待整改处数
- **拍摄要求**：对所有不合格项拍照记录，并附文字备注说明
