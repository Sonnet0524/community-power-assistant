## 快速参考

### 有效做法 ✅

1. **任务文件驱动** - 创建完整的任务文件，包含背景、要求、输出格式
2. **被动接收报告** - 不轮询Agent状态，等待Agent生成报告文件
3. **并行启动Agent** - 无依赖任务可并行启动，充分利用并行能力
4. **Test Team独立验证** - 独立测试可发现隐藏问题

### 无效做法 ❌

1. **主动轮询Agent状态** - 增加PM Team负担
2. **仅依赖Issue状态判断完成** - Issue可能被错误关闭
3. **忽略权限配置** - 非交互模式需要正确的权限设置
