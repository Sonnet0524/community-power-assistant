# PM Agent - 会话记录

> 📝 记录重要决策和关键交互

---

## 使用说明

本文件用于记录：
- 重要决策过程
- 关键问题和解决方案
- 与 Human 的重要讨论
- 项目里程碑

**记录时机**：
- 重要决策后
- 遇到重大问题时
- 完成里程碑时
- 发现新的最佳实践时

---

## 记录格式

```markdown
## YYYY-MM-DD HH:MM - [主题]

### 背景
[背景说明]

### 决策/问题
[决策内容或问题描述]

### 讨论过程
[讨论要点]

### 结论
[最终结论]

### 影响
[对项目的影响]
```
