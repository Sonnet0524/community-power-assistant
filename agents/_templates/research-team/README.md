# Research Team 模板

> 🔬 理论研究、框架设计、方法提炼

---

## 职责范围

- **理论研究**: Agent 协作模式、项目相关问题
- **框架设计**: 系统架构、协作流程、文档体系
- **方法提炼**: 从实践中提炼方法论
- **决策支持**: 多角度分析、质疑挑战

## 特点

- 不直接参与代码实现
- 不直接修改业务代码
- 纯研究型 Agent
- 站在框架层面思考

## 边界

**负责**:
- `docs/research/`
- `docs/methodology/`
- `docs/architecture/`

**不负责**:
- 业务代码
- 具体执行
- 配置修改

## 使用方式

```bash
# 复制模板
cp -r agents/_templates/research-team agents/my-research-team

# 定制 AGENTS.md
vim agents/my-research-team/AGENTS.md

# 配置并注册
# 详见 TEMPLATE-GUIDE.md
```
