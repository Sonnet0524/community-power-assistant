# Task-001: 基础环境搭建

## 任务概述

**任务ID**: TASK-001  
**任务名称**: 基础环境搭建  
**优先级**: 🔴 最高  
**预计工期**: 3-4天  
**依赖**: 无

## 任务目标

搭建Field Info Agent的开发、测试和部署基础环境，包括：
1. WPS开放平台账号和API权限
2. 企业微信自建应用
3. OpenClaw开发环境
4. 基础部署环境

## 详细工作内容

### 1. WPS开放平台准备

**工作内容**:
- [ ] 注册WPS开放平台企业开发者账号
- [ ] 完成企业认证
- [ ] 创建应用 "供电所信息管理系统"
- [ ] 申请以下API权限：
  - ksheet:read（查询多维表格）
  - ksheet:write（写入多维表格）
  - doc:read（读取文档）
  - doc:write（创建/编辑文档）
  - file:read（读取文件）
  - file:write（上传文件）
  - share:create（创建分享链接）
- [ ] 获取AppID和AppSecret
- [ ] 测试基础API调用

**交付物**:
- WPS应用配置文档
- API测试报告
- App凭证（安全存储）

### 2. 企业微信应用准备

**工作内容**:
- [ ] 在企业微信管理后台创建自建应用
  - 应用名称："现场信息采集助手"
  - 可见范围：供电所全员
- [ ] 配置应用功能：
  - 接收消息
  - 发送应用消息
  - 上传临时素材
  - 获取用户信息
- [ ] 配置回调URL：
  - URL: `https://your-domain.com/webhook/wecom`
  - 生成Token和EncodingAESKey
- [ ] 配置可信域名和IP白名单
- [ ] 创建群机器人（用于通知推送）
  - 名称："电小二通知助手"
  - 获取Webhook URL

**交付物**:
- 企业微信应用配置文档
- 回调配置信息
- 群机器人Webhook URL

### 3. OpenClaw开发环境

**工作内容**:
- [ ] 准备开发服务器（本地或云端）
- [ ] 安装Node.js 18+和npm/yarn
- [ ] 安装Docker和Docker Compose
- [ ] 部署Redis服务
- [ ] 安装OpenClaw CLI工具
- [ ] 创建项目目录结构
- [ ] 初始化OpenClaw项目

**交付物**:
- 开发环境配置文档
- 项目基础代码结构
- 环境验证测试报告

### 4. 基础设施部署

**工作内容**:
- [ ] 准备云服务器（4核8G推荐）
- [ ] 配置域名和SSL证书
- [ ] 配置Nginx反向代理
- [ ] 配置防火墙和安全组
- [ ] 部署Redis容器
- [ ] 准备生产环境配置

**交付物**:
- 部署架构图
- 服务器配置文档
- 网络配置说明

## 验收标准

- [ ] WPS API可正常调用（测试查询、插入操作）
- [ ] 企业微信应用可接收和发送消息
- [ ] OpenClaw环境可正常运行
- [ ] 所有密钥和凭证已安全存储
- [ ] 开发、测试、生产环境准备就绪

## 注意事项

1. **API权限申请可能需要1-3个工作日**，请提前申请
2. **企业微信回调URL需要公网可访问**，可先使用ngrok临时方案
3. **所有凭证需安全存储**，使用环境变量或密钥管理服务
4. **保留所有配置文档**，便于后续维护和交接
