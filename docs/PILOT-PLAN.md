## 🧪 测试用例

### 功能测试

#### TC-001: 配电房巡检流程
```
前置条件: 用户已在企业微信中
测试步骤:
  1. 用户发送"/start 阳光小区"
  2. 系统返回准备清单
  3. 用户发送"开始"
  4. 系统引导拍摄配电房照片
  5. 用户上传照片
  6. 系统自动分析铭牌信息
  7. 用户完成所有采集步骤
  8. 系统生成供电简报

预期结果:
  - 流程能正常完成
  - 照片成功上传并分析
  - 数据正确保存
  - 文档生成成功
```

#### TC-002: 应急事件响应
```
前置条件: 系统运行正常
测试步骤:
  1. 模拟停电故障
  2. 用户在群内报告
  3. 系统自动识别应急类型
  4. 系统推送应急方案
  5. 系统自动识别敏感客户
  6. 系统发送关怀消息

预期结果:
  - 应急响应时间 < 30秒
  - 应急方案准确
  - 敏感客户收到消息
```

## 📊 数据收集

### 系统数据

```sql
-- 每日统计脚本
SELECT 
    DATE(created_at) as date,
    COUNT(*) as total_tasks,
    COUNT(CASE WHEN work_type = 'power_room' THEN 1 END) as power_room_count,
    COUNT(CASE WHEN work_type = 'customer_visit' THEN 1 END) as visit_count,
    AVG(EXTRACT(EPOCH FROM (completed_at - created_at))/60) as avg_duration_minutes
FROM tasks
WHERE created_at >= CURRENT_DATE - INTERVAL '7 days'
GROUP BY DATE(created_at)
ORDER BY date DESC;
```
